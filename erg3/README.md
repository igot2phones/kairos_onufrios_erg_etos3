# Εργασία 3 — Ανάλυση δεδομένων ραδιοβόλισης

Το script `main.py` διαβάζει τα δύο αρχεία GeoJSON ραδιοβόλισης
(Θεσσαλονίκη και Αθήνα / Ελληνικό από το windy.com) και:

1. Σχεδιάζει για κάθε σταθμό ένα τριπλό γράφημα με τα κατακόρυφα προφίλ:
   - θερμοκρασίας ξηρού βολβού & σημείου δρόσου (°C),
   - μέτρου ταχύτητας ανέμου `|V| = sqrt(u² + v²)` (m/s),
   - σχετικής υγρασίας (%) — υπολογισμένης από τη σχέση Magnus.
   Ο τίτλος περιλαμβάνει το όνομα του σταθμού και την ημερομηνία/ώρα της
   ραδιοβόλισης, όπως προκύπτουν από το αρχείο.
2. Υπολογίζει προσεγγιστικά, με σταθερές αδιαβατικές θερμοβαθμίδες
   `Γ_d = 9.8 K/km` και `Γ_dew = 1.8 K/km`:
   - LCL = (T − Td) / (Γ_d − Γ_dew),
   - CCL ως την τομή της εκτεταμένης (με `Γ_dew`) επιφανειακής Td με το
     περιβαλλοντικό προφίλ θερμοκρασίας,
   - TCON = T(CCL) + Γ_d · h_CCL.

## Παραδοτέα

- `main.py` — το script.
- `16622-1776816000000-fm94thes.json` — μετρήσεις Θεσσαλονίκης.
- `16622-1776816000000-fm94ath.json` — μετρήσεις δεύτερης πόλης (Αθήνα, Ελληνικό).
- `thessaloniki.png`, `athina.png` — τα γραφήματα.
- `thessaloniki.txt`, `athina.txt` — αποτελέσματα υπολογισμών μαζί με τις
  τιμές που αναφέρει το windy.com.

## Εκτέλεση

```
python3 main.py
```

Απαιτούνται `numpy` και `matplotlib`.
