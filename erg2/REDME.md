# Εργασία 2 - Ανάλυση Μετεωρολογικών Δεδομένων

## Περιγραφή
Η εργασία υλοποιεί συλλογή, επεξεργασία και οπτικοποίηση μετεωρολογικών δεδομένων από:
1. OpenWeather API για Λάρισα και Βόλο.
2. Μετεωρολογικό σταθμό UTH από δημόσιο CSV.

Η βασική υλοποίηση βρίσκεται στο main.py.

## Στόχοι
- Σύγκριση θερμοκρασίας και αισθητής θερμοκρασίας μεταξύ δύο πόλεων.
- Υπολογισμός αισθητής θερμοκρασίας με Steadman 1984 για δεδομένα API.
- Ανάλυση ταχύτητας ανέμου και ριπών.
- Αντίστοιχη οπτικοποίηση για δεδομένα μετεωρολογικού σταθμού.

## Πηγές Δεδομένων

### OpenWeather API
- Περιοχή 1: Larisa,GR
- Περιοχή 2: Volos,GR
- Χρησιμοποιούμενα πεδία:
- temp
- feels_like
- humidity
- wind.speed
- wind.gust
- dt_txt

### Μετεωρολογικός Σταθμός UTH
- URL: https://greendigital.uth.gr/data/meteo48h.csv.txt
- Χρησιμοποιούμενα πεδία:
- Unix_time
- Temp_Out
- THW_Index
- Wind_Speed
- Hi_Speed

## Μεθοδολογία Υπολογισμού Αισθητής Θερμοκρασίας
Για τα δεδομένα API χρησιμοποιείται ο τύπος Steadman 1984:

AT = T + 0.33e - 0.70V - 4.0

Όπου:
- T: θερμοκρασία αέρα σε βαθμούς Κελσίου
- V: ταχύτητα ανέμου σε m/s
- e: τάση υδρατμών από θερμοκρασία και σχετική υγρασία
