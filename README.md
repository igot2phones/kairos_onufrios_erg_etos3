# Εργασίες Ονουφρίου - 3ο Έτος (2026)

Καλωσορίσατε στο αποθετήριο των εργασιών του μαθήματος Ονουφρίου για το 3ο έτος.
Εδώ συγκεντρώνεται οργανωμένα όλο το υλικό του 2026, με στόχο καθαρή δομή,
εύκολη πλοήγηση και συνεπή παρουσίαση κάθε εργασίας.

## Σκοπός

Το αποθετήριο λειτουργεί ως κεντρικό σημείο για:

- την παράδοση και αρχειοθέτηση των εργασιών,
- τη συνοδευτική τεκμηρίωση,
- τη σταδιακή εξέλιξη του κώδικα ανά ενότητα.

## Δομή

- `erg1/`: Πρώτη εργασία (κώδικας και αναλυτικό README).
- `erg2/`: Δεύτερη εργασία και σχετική τεκμηρίωση.

## Σημείωση

Η δομή και το περιεχόμενο εμπλουτίζονται προοδευτικά, ώστε κάθε εργασία να
παραμένει πλήρης, κατανοητή και εύκολα αξιοποιήσιμη.
